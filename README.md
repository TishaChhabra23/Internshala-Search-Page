# Internship Search Platform

A premium, modern internship search platform built with Next.js and React. Features a clean, highly responsive interface with smooth micro-interactions, live data integration, and a mobile-first design inspired by top tech startups.

## Features

- **Premium UI**: Minimalist aesthetic with soft shadows, rounded corners, and a highly polished layout.
- **Live Data**: Connects securely to Internshala to fetch and render live internship opportunities.
- **Dynamic Filters**: Sticky desktop sidebar and a floating mobile drawer that stay perfectly in sync.
- **Smooth Animations**: Seamless page transitions, hover states, and dynamic skeleton loaders powered by Framer Motion.
- **Responsive**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Components**: Headless UI, Lucide Icons
- **Language**: TypeScript

## Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/TishaChhabra23/Internshala-Search-Page.git
   cd Internshala-Search-Page
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
