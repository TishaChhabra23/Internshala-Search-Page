# Internshala Search Page

A premium, modern, and minimal frontend implementation for an internship search platform. Inspired by the clean aesthetics of Linear, Stripe, and Apple. Built with Next.js 15, React, TypeScript, and TailwindCSS 4.

## Features

- **Modern Architecture**: Next.js App Router with fully typed React components.
- **Premium Aesthetics**: Highly balanced spacing, pure white cards on an off-white background, and subtle soft shadows (`shadow-soft` to `shadow-hover`).
- **Smooth Microinteractions**: Utilizes Framer Motion for elegant hover elevations and a buttery-smooth mobile drawer.
- **Responsive Layout**:
  - **Desktop**: Sticky sidebar for filters and a wide main content area.
  - **Mobile**: App-like bottom slide-up drawer for filters.
- **Clean Typography**: Inter font implemented natively with `next/font/google`.
- **Reusable UI Components**: Clean abstraction with `cn` (Tailwind Merge + clsx) for scalable component design.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router, React 19)
- [TypeScript](https://www.typescriptlang.org/)
- [TailwindCSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/) (Icons)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/TishaChhabra23/Internshala-Search-Page.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
